Parágrafos e quebras de linhas

Não importa a quebra. Tudo vai aparecer normalmente no navegador.

Existem tags próprias para quebrar uma linha como
a tag <br>.

Vamos adicionar alguns símbolos especiais:
® © ™ € £ ¥ ¢ δ Δ ↑

Vamos adicionar emojis:
🎃 abóbora
👻 fantasma
O emoji aparece pelo código &#x + e código do emoji. Esse código pode ser encontrado no site emojipedia.
