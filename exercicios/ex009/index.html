Trabalhando com listas

Listas Ordenadas

A tag OL (ordered list) cria uma lista em que há uma sequência a ser seguida. A propriedade TYPE permite escolher o marcador que irá sinalizar a sequência. O valor "1" (2, 3, 4...) é o padrão, porém há também os valores "a" minúsculo (b, c, d...), "A" maiúsculo (B, C, D...), "I" romano (II, III, IV...) e i romano minúsculo (ii, iii, iv...)

  1. Acordar
    1. yoga
    2. arrumar a cama
  2. Tomar banho
    1. sabonete
    2. xampu
    3. hidratante
  3. Limpar o quarto
    1. cômoda
    2. mesa
    3. livros
  4. Ler por uma hora
    1. Americanah
    2. O cemitério dos vivos

Listas Não Ordenadas

A tag UL (unordered list) cria listas em que não há uma sequência de importância. Também tem a propriedade TYPE para modificar os marcadores. O valor "disc" (bola preenchida) é o padrão, mas também é possível mudar para os valores "circle" (bola vazia) e "square" (quadrado preenchido)

  • Nárnia
    • Anteontem
    • Ontem
    • Hoje
  • O cortiço
    • Meses atrás
    • Pretendo continuar
  • Pássaros na boca
    • Não sei se quero
    • Me interesso
  • O quarto de Giovanne

Listas de definição

A lista de definição é usada para listar termos e suas definições. A lista de definição usa três tags: a DL (Definition List, que engloba), a DT (Definition Term, o termo a ser definido) e a DD (Definition Description, a definição do termo). Tanto a tag DT e DD têm fechamento opcional.

HyperText Markup Languange, é a linguagem de marcação usada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites.
